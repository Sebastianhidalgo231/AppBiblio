cd Proyecto_Django
python -m venv venv
venv\Scripts\activate
pip install django
django -admin startproject proyecto_django

proyecto_django pasa a ser src

cd src

Para hacer correr el servidor 

python manage.py runserver

python manage.py migrate

Buscar SQLite entre las extensiones

En el buscador de VSCode escribir SQLite Open Database y hacer clic sobre la línea del proyecto

python manage.py createsuperuser

Creamos usuario y contraseña 

Escribir "localhost..." y luego "/admin"

Cambiar horario e idioma en settings.py

Por razones de seguridad, DEBUG tiene que quedar en True antes de entrar a producción.  

Crear carpeta "media" en src. Es donde estarán los archivos multimediales. También creamos 
una carpeta static en src. 

Para crear la aplicación 

python manage.py startapp nombre_aplicacion

Escribir "nombre_aplicacion" en INSTALLED_APPS de settings.py. 

En models.py es donde estarán las clases. 

Después de definir una clase, escribir en la terminal: 

python manage.py makemigrations

Así queda definido un modelo. Luego: 

python manage.py migrate

Escribir en settings.py:

STATICFILES_DIR = [
    BASE_DIR / "static",
]

Los Templates son plantillas que se van a convertir en páginas web. Ingresar: 

"DIRS": [
    BASE_DIR / "templates",
]

Views: Lo que queremos hacer
Templates: Lo que queremos mostrar
URLs: Cómo vamos a hacer para llegar
